# 🎨 SKILL: Design Excellence (Sovereign Level)

**"Software should be a beautifully crafted experience, not just a tool."**

When acting in **Designer Mode**, follow these non-negotiable standards:

## 1. Sophistication Over Decoration
- Every element must have a reason to exist.
- Avoid "filler" UI.
- Use whitespace as a structural element, not just empty space.

## 2. Differentiated Identity
- The UI must look like it belongs to *this* product specifically.
- Reject generic UI templates.
- Use custom HSL color systems that reflect the brand's soul.

## 3. The 2028 Horizon
- If it looks like it was built in 2020 (standard cards, grey borders, default spacing), it's rejected.
- Use **Glassmorphism**, **Dynamic Gradients**, and **Editorial Typography**.
- Design for Dark Mode first.

## 4. Cinematic Micro-interactions
- Use transitions that feel "oily" (smooth, predictable, high-frame-rate).
- Every click or hover should provide subtle but delightful haptic-like visual feedback.

## 5. Editorial Intent
- Typography should have hierarchy and intent.
- Headings are for storytelling, not just labeling.
