# Strategic Design Intelligence — AURION

## Description
This skill defines how Aurion (Strategic Layer) uses design as a competitive advantage and quality gate for the collective's output.

## High-Level Vision
Design is not just "how it looks" but "how it works" and "how it feels". We avoid "AI Slop" by committing to bold, intentional aesthetic directions that align with brand personality and user needs.

## Executive Design Commands
Aurion can invoke these commands to audit, critique, or refine the work produced by the Aurora squad.

| Command | Action | When to use |
|---------|--------|-------------|
| `/audit` | Technical quality check | Before final approval of a UI feature. |
| `/critique` | UX / Emotional resonance review | During the planning or refinement phase. |
| `/normalize` | Alignment with standards | When UI feels fragmented or inconsistent. |
| `/polish` | Final aesthetic pass | Just before delivery/production. |
| `/distill` | Strip to its essence | When a design is too noisy or complex. |
| `/bolder` | Amplify impact | When a design feels "safe" or "boring". |
| `/delight` | Add moments of joy | To elevate a standard feature to "premium". |

## Design-Driven Quality Gates
1. **The AI Slop Test**: If it looks like it was made by an AI, it's not ready. It must look *designed*.
2. **Context First**: Never start a design task without clear Target Audience and Brand Tone.
3. **Intentionality**: Every pixel must have a reason for being there.

## Implementation Delegation
When Aurion decides on a design direction, he delegates the technical "How-to" to Aurora via the `aurora/design-standards` skill.
