**Aurion Nexus** is a state-of-the-art multi-agent collective designed for autonomous software engineering, strategic architectural management, and premium UI/UX delivery. It transforms the interaction with AI from a command-based chat into an **intent-driven partnership**.

---

## 🧠 The Core Philosophy: "Intelligent Autonomy"

Aurion Nexus is governed by a formal **Soul** and **Executive Rules** framework, ensuring that every line of code is not just functional, but enterprise-grade, secure, and beautiful.

- **Agent-First**: Work is automatically dispatched to a specialized squad of 11+ agents.
- **Intent-Based**: You speak your goal; the system assembles the squad and selects the protocol.
- **Enterprise Rigor**: Mandatory TDD (Test-Driven Development), Security Audits, and ADR (Architecture Decision Records).

---

## 👥 The Squads

### 🏛️ Aurion (The Strategic Commander)
The "Brain" of the operation. Aurion handles requirements, trade-offs, and high-level design.
- **Chief of Staff**: Coordinates complex missions and agent assembly.
- **Socratic Deep Interviewer**: Eliminates ambiguity before execution starts.

- **Impeccable Designer**: Ensures premium aesthetics (no "AI Slop").
- **Database/Performance Specialists**: Optimizes for scale and speed.

## ✨ Key Features

- **The Ralph Loop**: An autonomic persistence cycle (Plan → Exec → Verify → Fix) that doesn't stop until the mission objective is achieved.
- **Impeccable Design Standards**: Built-in design system tokens for modern, vibrant, and sleek interfaces.
- **The Chronicle**: A persistent project diário that stores historical context, preventing "session amnesia".
- **Zero-Friction Interface**: Proactive routing that understands natural language intent.

## 🛠️ The Sovereign CLI

Manage your entire collective with a single command based on the `bin/aurion` entrypoint.

```bash
./bin/aurion health   # Run core diagnostics
./bin/aurion pulse    # Generate context snapshot (PULSE.md)
./bin/aurion status   # View current mission from the Chronicle
./bin/aurion setup    # Re-register all agent skills
```

## 🚀 Quick Start


See [INSTALL.md](INSTALL.md) for a detailed setup guide.

1. Initialize the core: `./bin/aurion setup`.
2. Configure your keys in `secrets/`.
3. Open your AI Assistant (like Claude Code or Cursor).
4. Say: *"Aurion, make a new CRM dashboard"* and watch the squad take over.
