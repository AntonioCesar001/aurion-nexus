# Contributing to Aurion Nexus

First off, thank you for considering contributing to Aurion Nexus! It's people like you who make this a great sovereign agentic framework.

## Code of Conduct
By participating in this project, you agree to abide by our standards of technical excellence and transparent documentation.

## How to Contribute
1.  **Fork the repo** and create your branch from `main`.
2.  **Ensure SEIP v2 compliance**: Every architectural change must come with an ADR.
3.  **Tests are non-negotiable**: We target 80% coverage.
4.  **Conventional Commits**: Use `feat:`, `fix:`, `docs:`, `chore:`, etc.
5.  **Audit First**: Run `./bin/health-check.sh` before submitting.

## Pull Request Process
1.  Update the `CHRONICLE.md` with your changes.
2.  If you add a new capability, create a corresponding Skill in `.agent/skills/`.
3.  The maintainers will review and merge after CI passes.

Stay sovereign.
